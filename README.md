# 不動産投資分析システム

物件チラシ画像を入れるだけで、買うべきか判断できるレポートを自動生成するシステムです。

## セットアップ

```bash
# 依存パッケージインストール
pip install -r requirements.txt

# APIキー設定
export ANTHROPIC_API_KEY="your-api-key"
```

## 使い方

```bash
# 1. input/ に物件チラシ画像を配置
# 2. 実行
python run.py
# 3. output/ にレポートが生成される
```

対応画像形式: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`

## 出力内容

### PDFレポート
1. エグゼクティブサマリー（買い/見送りの判断）
2. 物件概要
3. 賃貸 vs 民泊 収益比較
4. 個人 vs 法人 税務比較
5. 融資戦略（5金融機関比較）
6. リスク分析
7. 出口戦略（5/10/15/20/25/30年）
8. 感度分析

### Excelレポート
- サマリー
- 35年キャッシュフロー
- 融資比較
- 出口戦略
- リスク分析

## 分析指標

| 指標 | 説明 | 目安 |
|------|------|------|
| 表面利回り | 年間賃料 / 物件価格 | 7%以上 |
| 実質利回り | NOI / 物件価格 | 4%以上 |
| DSCR | NOI / ローン返済額 | 1.2以上 |
| CCR | 税引前CF / 自己資金 | 4%以上 |
| IRR | 内部収益率 | 5%以上 |

## 対応金融機関

| 金融機関 | 金利 | 特徴 |
|----------|------|------|
| メガバンク | 1.5-2.5% | 審査厳格、民泊NG |
| 地方銀行 | 1.8-3.0% | 柔軟、地元優遇 |
| 信用金庫 | 2.0-3.5% | 最柔軟、小口OK |
| 日本政策金融公庫 | 1.0-2.5% | 事業計画重視、民泊OK |
| ノンバンク | 3.0-5.0% | 審査緩い、民泊OK |

## フォルダ構成

```
property-analyzer/
├── input/           ← 物件画像を配置
├── output/          ← レポート出力先
├── data/            ← 抽出データ保存
├── src/
│   ├── extractor.py      # 画像→物件情報抽出
│   ├── analyzer.py        # 収支分析・指標計算
│   ├── financing.py       # 融資シミュレーション
│   ├── tax_compare.py     # 個人vs法人比較
│   ├── risk.py            # リスク・出口戦略
│   └── report_generator.py # PDF/Excel生成
├── run.py           ← メイン実行
├── config.yaml      ← 設定（金利等カスタマイズ可）
└── requirements.txt
```

## 設定カスタマイズ

`config.yaml` で以下を調整可能:
- 金融機関の金利・条件
- 空室率・管理費率
- 税率
- 感度分析パラメータ
- 出口戦略の分析年数
