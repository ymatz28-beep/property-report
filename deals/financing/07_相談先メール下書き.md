## 2. 福岡市 保健所（中央区）— 簡易宿所 許可の事前相談【電話／窓口】

> 旅館業の事前相談は電話・窓口が基本。下は「聞くことリスト」。窓口は福岡市 中央区保健所 衛生課（旅館業担当）。
> 参考: https://www.city.fukuoka.lg.jp/hofuku/hokensho/kurashinoeisei/jigyosya/ryokan/ryokangyounoeigyounituite.html

**電話で名乗る一言**
「中洲2-1-11の区分マンション1室で簡易宿所（旅館業）の開業を検討しており、事前相談をお願いしたいです。」

**聞くことリスト**
```
1. 区分マンション1室を簡易宿所にする場合の、構造・面積・換気・便所・洗面の要件
2. 玄関帳場（フロント）はICT代替（タブレット＋顔認証＋10分駆けつけ＋防犯カメラ）で可能か。その具体要件
3. 消防法令適合通知書の取り方と、1室を宿泊用途にした場合の建物全体（複合用途化）への波及
   — 共用部の感知器追加や他住戸同意が要るケースの有無。消防署への事前相談の進め方
4. 用途変更（200㎡未満で建築確認不要だが適合義務は残る）について実務上の注意
5. 申請から許可までの標準的な審査期間と、申請に必要な図面・書類一式
6. 改装・消防工事と許可申請の前後関係（どこまで工事してから申請するのが通常か）
```

## 3. 福岡商工会議所 等 認定支援機関 — 経営力向上計画の相談【メール／電話】

> 経営強化税制（即時償却 or 10%税額控除）の入口。設備取得の"前"に計画認定が必要。
> 認定支援機関は商工会議所・税理士・金融機関など。まずは福岡商工会議所の経営支援窓口へ。
> 参考: 中小企業庁 経営強化税制 https://www.chusho.meti.go.jp/keiei/kyoka/kyoka_zeisei.html

**件名**: 経営力向上計画の認定についてのご相談（宿泊業の設備投資・中小企業経営強化税制）

```
福岡商工会議所 経営支援ご担当者様

お世話になります。iUMAプロパティマネジメント合同会社（代表社員 手嶋耕一）と申します。
福岡市博多区中洲で簡易宿所（旅館業）の開業を準備しており、宿泊運営の設備投資（スマートロック・
予約管理システム・家具家電・空調等、概算190万円）について、中小企業経営強化税制（即時償却 or
10%税額控除）の活用を検討しています。

つきましては、その前提となる経営力向上計画の認定について、下記をご相談させてください。

1. 当社（不動産賃貸業＋これから宿泊業）が経営力向上計画の対象になるか
2. 認定までの標準的な期間と、設備取得の前に認定を取るための逆算スケジュール
3. A類型（生産性向上）／B類型（収益力強化）の選択と、B類型の税理士事前確認書の段取り
4. 認定支援機関としてのご支援可否、または適切な窓口のご紹介
5. あわせて、固定資産税の軽減（先端設備等導入計画）も検討余地があれば概要

設備の発注時期を誤って税優遇を逃さないよう、順序を確定させたく存じます。
```
